# Cashflow frontend

React + TypeScript + Vite. Run the app from the **repository root** (see root [`README.md`](../README.md)):

```bash
yarn dev
# or
npm run dev
```

This starts the API and Vite together. To run only the frontend (API must already be on port 3001):

```bash
yarn workspace frontend dev
# or
npm run dev -w frontend
```
